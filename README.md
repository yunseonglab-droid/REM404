Project Overview
REM404 is a WebAR-based archive that allows visitors to scan a physical photograph and leave anonymous memories connected to that place. Instead of simply displaying augmented reality content, the project transforms photography into an interactive archive where individual memories become part of a collective experience.
The project combines MindAR, A-Frame, Firebase, and a lightweight WebAR interface to provide an installation that works directly in a mobile browser without requiring an application.

Why This Project Is Open Source
REM404 is released as an open-source project because I believe interactive media art should be easier for artists, students, and museums to build upon.
Many photography students and exhibition creators want to experiment with WebAR but often struggle with complex setup processes or expensive proprietary software.
By making REM404 open source, I hope others can:
* learn from a real-world WebAR project,
* adapt it for exhibitions,
* improve accessibility,
* contribute new interaction ideas,
* and expand the project into different artistic contexts.
