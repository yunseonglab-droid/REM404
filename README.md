# Long-term Vision

REM404는 단순한 WebAR 프로젝트가 아니다.

이 프로젝트는 「남겨진 사람들의 기억과 흔적」 작업을 위한 인터랙티브 아카이브 플랫폼을 연구하는 과정이며, 사진·AI·WebAR·인터랙션을 결합한 새로운 전시 경험을 만드는 것을 목표로 한다.
